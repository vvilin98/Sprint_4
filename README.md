# Sprint_4
Проект по автоматизации тестирования сервиса **ЯндексСамокат**
Стендом для тестирования является - [ЯндексСамокат](https://qa-scooter.praktikum-services.ru/)

## Общая информация по репозитори.
### utils:
- locators.py  | Локаторы 
- test_data.py | Тестовые данные пользователей
- urls.py      | Используемые urls

### tests:
- test_order_page.py | Тест страницы оформления заказа 
- test_home_page.py  | Тест домашней страницы 
- test_FAQ           | Тест "Вопросы о важном"

### pages:
- base_page.py | Базовые действия
- home_page    | Действия на домашней странице
- order_page   | Действия при заказе самота

### Результаты тестирования 
- Рез-ты лежат в директории allure_results
- Посмотреть отчет в веб версии, находясь в корневой папке проекта, командой в терминале : ```allure serve allure_results```

### Фреймворк
- pytest + selenium + allure
- Внешние зависимости лежат в requirements.txt
- Перед работай с данным кодом требуется ряд библиотек, установить в терминале командой:  ```pip3 install -r requirements.txt```
